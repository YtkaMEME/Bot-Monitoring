Руководство пользователя Telegram Bot для анализа анкет

1. Начало работы
   - Найдите бота в Telegram по его имени
   - Отправьте команду /start для начала работы
   - Бот поприветствует вас и предложит отправить файл с данными анкеты

2. Подготовка файла с данными
   - Файл должен быть в формате Excel (.xlsx)
   - Данные должны быть структурированы в виде таблицы
   - Первая строка должна содержать заголовки вопросов
   - Каждая строка представляет ответы одного респондента

3. Процесс анализа
   а) Отправка файла:
      - Нажмите на значок скрепки в чате
      - Выберите "Документ"
      - Найдите и отправьте ваш Excel-файл

   б) Настройка параметров:
      - Укажите количество участников опроса
      - Ответьте на вопросы о наличии специальных вопросов:
        * Есть ли вопросы по настроению?
        * Есть ли вопросы NPS?
        * Есть ли вопросы CSI?

4. Получение результатов
   - Бот обработает данные и отправит вам:
     * Обработанный Excel-файл с результатами
     * CSV-файл с данными
     * Краткое описание результатов

5. Основные команды
   - /start - Начать работу с ботом
   - /cancel - Отменить текущую операцию
   - /get_my_id - Получить свой Telegram ID
   - /help - Получить справку

6. Типы поддерживаемых вопросов
   - Шкальные вопросы (например, от 1 до 10)
   - Вопросы с одиночным выбором
   - Вопросы с множественным выбором
   - Матричные вопросы
   - Трехмерные матричные вопросы
   - Свободные текстовые ответы
   - Специальные вопросы (NPS, CSI)

7. Советы по использованию
   - Убедитесь, что файл не открыт в других программах
   - Проверьте формат данных перед отправкой
   - Дождитесь полной обработки файла
   - Сохраняйте полученные результаты

8. Решение проблем
   Если бот не отвечает:
   - Проверьте подключение к интернету
   - Убедитесь, что файл не слишком большой
   - Попробуйте отправить команду /start
   - Если проблема сохраняется, обратитесь к администратору

9. Ограничения
   - Максимальный размер файла: 20 МБ
   - Поддерживаемые форматы: .xlsx
   - Количество строк: до 100,000
   - Количество столбцов: до 100

10. Конфиденциальность
    - Все данные обрабатываются локально
    - Файлы удаляются после обработки
    - Результаты доступны только вам
    - История обработки не сохраняется

11. Дополнительная информация
    - Для получения доступа к боту обратитесь к администратору
    - Администратор добавит ваш Telegram ID в список разрешенных пользователей
    - После этого вы сможете использовать все функции бота

12. Обновления
    - Бот регулярно обновляется
    - Новые функции добавляются автоматически
    - Следите за сообщениями о обновлениях
    - При появлении новых функций вы получите уведомление
